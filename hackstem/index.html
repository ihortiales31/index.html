HACKSTEM 2023

- Objetivos General -

Diseñar un sistema de monitoreo para el estudio y alerta temprana de episodios de muerte súbita cardiaca en adultos mayores.

- Objetivos Especificos -

- Entrevistar a un experto médico para identificar los síntomas y dispositivos existentes.

- Utilizar la información obtenida para elaborar una lista con los requerimientos y características del sistema.

- Establecer la arquitectura del sistema.

- Establecer un método óptimo de comunicación entre el software y hardware del sistema de monitoreo.

- Desarrollo -

Propusimos un sistema el cual se basa en circuito en una playera de compresion en la cual se encontrarian 3 dispositivos: electrocardiograma, pulsioximetro y termometro corporal. El microcontrolador del sistema seria un ESP32, el cual mandaria la información a un servidor para el procesamiento de datos.

El procesamiento de datos seguiria la siguiente ruta, donde en caso de que los datos obtenidos cumplan con los parametros de alerta, se comunicara con el doctor de cabecera y un familiar para la rapida atención.

- Conclusiones -

El sistema tiene como objetivo mejorar la calidad del tratamiento médico al monitorear la salud de los pacientes de forma remota debido a que los equipos actuales para estudios de este tipo son poco ergonomicos y molestos para los pacientes.
