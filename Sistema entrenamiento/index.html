Implementación de un sistema de entrenamiento para niños con discapacidad visual

- Objetivos General -

Implementar un sistema de entrenamiento que ayude al desarrollo de la psicomotricidad gruesa en niños con discapacidad visual del equipo “Topos Puebla”

- Objetivos Especificos -

- Definir los requerimientos y características técnicas de un traje que ayude al desarrollo psicomotriz en niños con discapacidad visual

- Diseñar un sistema que ayude al desarrollo de la psicomotricidad gruesa de acuerdo con los requerimientos

- Desarrollar elementos de hardware y software del sistema entrenamiento.

- Realizar pruebas de campo con usuarios de la comunidad Topos Puebla.

- Desarrollo -

Diseñamos una placa PCB que tuviera los siguientes componentes: ESP32WROOM como microcontrolador, un modulo de vibración PWM para retroalimentación haptica, un acelerometro MPU-6050 para obtener angulos de inclinación y pines de programación. Tambien se diseño una caja para proteger la placa y que se pudiera colocar con una correa en las extremidades de nuestros usuarios.

A continuación se muestra la placa y su caja, con una correa de velcro para ajustarla en la extremidad deseada y los cables de alimentación.

Una vez que se armaron las 8 correas con sus placas y que estuvieron conectadas a la fuente de poder, se coloco el traje en nuestro usuario control para obtener los angulos estandar y determinar el rango permisible. Para la medición de los angulos, se uso comunicación Wifi para mandar los datos de la ESP32 a un servidor donde se guardaban los datos y graficaban en tiempo real.

Por ultimo, se hicieron las pruebas correspondientes con los usuarios de Topos Puebla.

- Conclusiones -

El sistema tiene como objetivo mejorar la calidad del tratamiento médico al monitorear la salud de los pacientes de forma remota debido a que los equipos actuales para estudios de este tipo son poco ergonomicos y molestos para los pacientes. Durante la investigación del proyecto observamos la poca investigación que se ha hecho en este campo, siendo una gran oportunidad para seguir trabajando este traje. Tambien se tuvo un focus group con los usuarios para obtener retroalimentación sobre las areas de mejora del primer prototipo.
